DIASPORA* EASYSHARE

Firefox Addon

INFO Latest version: 0.12.2
# Getting started
Once installed enter URL and credentials of your diaspora* pod in the addon preferences.
# Preferences
  • Pod Url : Enter the full address (https://yourpod.tld) of your pod. (REQUIRED)
  • Credentials : Enter and save username and password to access your pod. (REQUIRED)
    The credentials are securely stored in the Firefox Password Manager. You can manage them opening about:preferences#security and clicking the 'Saved Passwords' button.
  • Bitly OAuth : Start the OAuth web flow to authorize Easyshare to shorten URLs with your Bitly account. (Required to shorten URLs)
  • Provider : select to display the status message provider in the post. (via Easyshare).
# The popup dialog
  • Short URL (button) : to shorten long URLs with Bitly (prior authorization).
  • Markdown (buttons) : to add markdown syntax for italic, bold or quotation. To the entire text or by selecting part of it.
  • Share with : input box to enter an aspect name (to share with a single aspect), all_aspects (to share with all your aspects) or public (to share public).

# Tips
  • You can leave the 'Share with' box blank to share public.
  • All textarea elements in the popup dialog are resizable to better view their content. To bring them back to their default size just double click on the resizer.
  • The markdown's buttons for italic and bold they skip adding syntax if the text contains newlines. In this case to add the syntax you need to select each block of text without newline.
  • Always use backslash escapes to generate the literal characters used by markdown syntax: *_> (e.g. diaspora\*).
