DIASPORA* EASYSHARE

Firefox Add-on

Latest version: 0.14.2
# About migration to WebExtensions
Version 0.14.1 is a major update to migrate the add-on to the WebExtensions APIs and allow it to work with Firefox 57 Quantum.
This update makes Easyshare a bit different. Some features are added and other are no longer there.
The most relevant differences are:
  1. the panel looks like the publisher used by diaspora*. Better use of markdown and ability to preview the post.
  2. The panel isn't positioned at center of the page but attached to the toolbar, below the icon. (*)
  3. It is no longer possible to shorten URLs by linking your Bitly account.
  4. It is no longer possible to display the status message provider in the post.
  5. It is currently in english. No localized strings for other languages.
(*) With Firefox 56 the panel not opens automatically when you click the context-menu item. You have to click the toolbar icon that becames red when the content is inside. This will no longer have to be done with Firefox 57 and above.
# Usage
Usage is very simple. After installed enter pod URL and credentials in the Preferences. And go, right click on images, text selection or videos, to share the content you want.
Note.
On videos you have to right click twice to show the browser's context-menu.
Tags must be separated by commas and can be entered with or without the hash symbol.
