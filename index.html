DIASPORA* Easyshare

Mozilla Firefox add-on to easily share web content on your Diaspora* pod.
https://addons.mozilla.org/firefox/addon/diaspora-easyshare/

Settings

After install you must save the URL of your Diaspora* pod.
Go to Firefox > Addons > Extensions > Diaspora* Easyshare > Preferences
and enter the URL (e.g. https://yourpod.com).

Note:
There's a little tedious bug when typing the url (Bug 784335):
Typing forward slash ("/") in the preference field moves cursor to search field. You need to put again the cursor in the field.

Usage

- Share images
Right click on images and select "Share on Diaspora*".
- Share videos
You can share videos from youtube.com. Right click on video page and select "Share video on Diaspora*".
You must be logged to your Diaspora* pod to use this addon.

The panel

- Title
Shows document-title/video-title. You can edit or change it.
- Url
Shows document-URL/video-URL. You can shorten long urls.
- Text
Additional text. Automatically shows text selected (images) or description (videos). You can add markdown emphasis (italics) or use your own markdown syntax.
- Tags
Hashtags. Enter them separated by commas.

The post

Diaspora* Easyshare send the post to Diaspora via the publisher bookmarklet, forcing it to build the post as /image/title/link/text/hashtags/ for images and /title/text/hashtags/link/video/ for videos.

Preferences

Firefox > Addons > Extensions > Diaspora* Easyshare > Preferences
- Advertises
Add/remove "via Easyshare" at end of post.
- Set the publisher window height
If you have many aspects you can increase the publisher window height.
- Your Bitly API credentials
Uses your own Bitly API credentials for shorten URLs or leave the fields blank if you want use the internal ones.
