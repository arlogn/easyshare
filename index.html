DIASPORA* Easyshare

Mozilla Firefox add-on to easily share web content on your Diaspora* pod.

Diaspora* Easyshare extends your right-click context menu with the ability to share images, text and videos.

After installing the add-on you must enter the URL of your Diaspora* pod in the Options/Preferences.

Choose what to share with the right-click context menu.

Share images
Right click on images and choose Share image on Diaspora*.
Selected text in the page will be automatically entered in the panel Text field.

Share only selected text
Right click on selected text and choose Share selection on Diaspora*.

Share YouTube videos
Right click on youtube video page and choose Share video on Diaspora*.

You must be logged to your Diaspora* pod to use this add-on.

The panel fields.

Title
Document title / video title. To the title string will be automatically added markdown syntax for strong emphasis. So it's not recommended to use markdown here.

Url
Document URL / video URL. With a click you can shorten long URLs.

Text
The body of the post. It shows (if any) selected text or video description. With a click you can format entered text adding emphasis, strong emphasis or

defining an indented quoted section.
You can do that on the whole text or only on a selection. Or both.
Alternatively you can manually enter your own markdown sintax.

Tags
Hashtags separated by commas (tag1,tag2,tag3...).

Easyshare sends the post to Diaspora* via the publisher for bookmarklet.
The content is structured in this way:

image / title / url / text / hashtags

title / text / hashtags / url / video

about:addons > Diaspora* Easyshare > Options/Preferences.

Pod URL (required)
The URL of your diaspora* pod.

Publisher Width
Set the publisher window width. Default 600 (optimal).

Publisher Height
Set the publisher window height. Default 250 (increase it if drop-down menu for choose aspects isn't entirely visible).

Bitly API credentials (Login/API key)
Use your own Bitly API credentials to shorten URLs or leave these fields blank to use the internal ones (rate limited).

Multi-language localization. Currently, in addition to the default (en-US), is localized for these languages:

- Italian (it-IT)
- Dutch (nl-NL) (thanks to @meneer)

If you want to add localization for your language open an issue on GitHub. Or create your local translation file in the /local folder and push a commit. Any improvements for the english language are welcome.

Use GitHub to report bugs, issues or suggestions. Thanks.
