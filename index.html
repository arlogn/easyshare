DIASPORA* Easyshare

Mozilla Firefox add-on to easily share web content on your Diaspora* pod.

Diaspora* Easyshare extends your right-click context menu with the ability to share images, text and videos.

After installing the add-on you must save the URL of your Diaspora* pod in the Preferences.

Note:
There's a little tedious bug (784335) when typing the url in an add-on preference field:
Typing forward slash ("/") in the preference field moves the cursor in the Search field. To continue typing you need to put the cursor back in the preference field.

Choose what to share via the right-click context menu.

Share images
Right click on images and choose 'Share image on Diaspora*'.
Selected text in the page will be automatically entered in the panel Text field.

Share only selected text
Right click on selection and choose 'Share selection on Diaspora*'.

Share YouTube videos
Right click on youtube video page and choose 'Share video on Diaspora*'.

You must be logged to your Diaspora* pod to use this add-on.

The panel fields from which to build the post

Title
This field shows the document title or video title. You can change or modify its content. To the title string will be automatically added markdown syntax for strong emphasis. So it's not recommended to use markdown here.

Url
This field shows the document URL or video URL. With a click you can shorten long URLs.

Text
This field contains the body of the post and you can enter what you want. It shows (if any) selected text or video description. With a click you can add markdown syntax to give emphasis to the text(*) or use your own markdown syntax.
(*) If text does not contain underscores in URLs. All underscores and asterisks will be escaped so URLs would be modified.

Tags
Field to enter hashtags separated by commas (tag1,tag2,tag3...).

Easyshare sends the post to Diaspora* via the publisher bookmarklet, forcing it to structuring the post in this way:
image / title / link / text / hashtags / --- when you share images
title / link / text / hashtags / --- when you share text-only
title / text / hashtags / link / video / --- when you share videos.

By default Easyshare puts a widget in the browser add-on bar. Clicking the icon you can quickly send text-only posts.
The widget can be permanently removed by uncheking the checkbox in the Options/Preferences.

Type about:addons in the Firefox address bar then Diaspora* Easyshare > Options/Preferences

Pod URL (required)
The URL of your diaspora* pod.

Widget
Show/remove the widget in the browser add-on bar.
After removing it, if you select the checkbox to show it again in the same Firefox session, you must restart the browser.

Advertises Easyshare
Add/remove 'via Easyshare' at the end of post.
Adding 'via Easyshare' at the end of post as well as an advertisement for this add-on it's a necessity. It's a way to utilize the hyphen that is automatically entered by the bookmarklet (all params passed to the bookmarklet are separated by an hyphen). If you choose not to advertise the add-on, the hyphen will be positioned at the end of hashtags.

Publisher window height
Increase the publisher window height. This is useful if the Aspects drop-down menu isn't entirely shown.

Bitly API credentials (Login/API key)
Use your own Bitly API credentials to shorten URLs or leave these fields blank to use the internal ones.

From version 0.5.2 Easyshare supports multi-language localization. Currently, in addition to the default (en-US), is localized for these languages:

- Italian (it-IT)

If you want to add localization for your language open an issue on GitHub.

Please, use GitHub Issues Tracker to report bugs, issues or features requests. Thanks.
