DIASPORA* EASYSHARE

Firefox add-on

Extend the right-click context menu with the ability to share web content on your diaspora* pod.
Edit and preview the post in a publisher which popup from the browser toolbar.

Latest version: 0.16.0
install (require Firefox 57+)

Getting started

  • Preferences
  • After installing the add-on you have to enter URL and credentials of your pod in the Preferences.
  • Get your aspects
  • The dropdown menu to select aspects initially contains only Public and All-aspects. Use the button Get Aspects to update it with all your aspects.

Usage

  • The conventional use is via the context menu item (Share this content) that allows you to add the markdown representation of the content chosen within the publisher. But you can also simply open the publisher and enter whatever you want.
  • The context menu
  • The list of contexts in which the menu item will appear:
    • Image
    • Displayed when you context-click an image in the page. Note:
      if the image is wrapped with an anchor, the link is preserved.
      Keep in mind that the menu item only appears on inline images, not on images added via css or javascript.
    • Selection
    • Displayed when you context-click a selection in the page. This is generally used to select text, which can be added whitin the publisher along with other content. Some examples:
      - select some text and context-click an image
      - select some text and context-click a video
      - select some text and context-click a link.
    • Link
    • Displayed when you context-click a link in the page. Note:
      this is the only element added without the page title and the page URL.
    • Audio/Video
    • Displayed when you context-click html5 audio/video elements in the page. Note:
      diaspora has added support for embedding html5 audio/video from version 0.7.4.0.
      Also note that this fails on html5 videos from Youtube because its player dynamically construct the stream. Look at the note below to know on how to pass the video URL from Youtube.
    • Page
    • Displayed when you context-click the page. This refers to click on any place in the page but not on other contexts. Allows you to add the page title and page link within the editor. Note:
      this can also be used to share all those resources — from media sharing sites as Youtube, Vimeo, Soundcloud or Twitter — that are embedded posting their URLs.
      What to click in this pages:
      - Youtube: in the video page, context-click anywere but not on the video or other contexts.
      - Vimeo: in the video page, context-click the video.
      - Soundcloud: in the track page, context-click the player.
      - Twitter: open a tweet and context-click anywere but not on other contexts.
  • The tags input box
  • This box permits to enter tags at the end of the post in a simpler and faster way. You can enter them without the hash and with or without spaces between them, simply by separating them with a comma.
    But of course, you can also enter tags directly within the editor.
