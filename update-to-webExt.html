DIASPORA* EASYSHARE

Updated to version 0.14.1

Changes:

  • Migrated to WebExtensions APIs

About this updateThis is a major update to migrate the add-on to the WebExtensions APIs and allow it to work with Firefox 57 Quantum. Some days ago I uploaded on AMO the update 0.13.2 to inform that Easyshare it would not work with Firefox 57. It had to be the death of Easyshare.

Well, I changed my mind.

This update makes Easyshare a bit different because the code was written for what it was supposed to be a new add-on. Some features are added, other are no longer there. The most relevant differences are:

  1. the panel looks like the publisher used by diaspora*. Better use of markdown and ability to preview the post.
  2. The panel isn't positioned at center of the page but attached to the toolbar, below the icon. (*)
  3. It is no longer possible to shorten URLs by linking your Bitly account.
  4. It is no longer possible to display the status message provider in the post.
  5. It is currently in english. No localized strings for other languages.

(*) Note
With Firefox 56 the panel not opens automatically when you click the context-menu item. You have to click the icon that becames red when the content is inside. This because the api to open the panel programmatically is available only in Firefox 57.

You have to re-enter all data in the Preferences!
